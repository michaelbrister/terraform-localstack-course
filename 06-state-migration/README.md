
# Lab 06 — Migrating Terraform State (Local → S3)

## What this lab teaches you
This lab walks through **state migration**, one of the most dangerous Terraform operations.

Concepts covered:
- `terraform init -migrate-state`
- Backend configuration changes
- State safety

## What to observe
- Terraform detects backend changes
- State is moved, not recreated

## Exam notes
State migration questions are common and often subtle.
