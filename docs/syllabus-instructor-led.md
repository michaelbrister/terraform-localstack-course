
# Instructor-Led Syllabus

## Option 1: 2-day bootcamp (fast)
**Audience:** existing engineers, limited Terraform

Day 1
- Mental model + workflow (Labs 01–03)
- Console drills (08)
- Dynamic blocks (04)
- Begin state backends (05)

Day 2
- State migration + lock drills (06, 13)
- Modules + refactors (07, 09)
- Break/fix guided drills (10)
- CI concepts: policy gates + tests (11, 16)
- Homework: timed drills (12)

## Option 2: 5-day program (recommended for mixed skill levels)
Day 1: 00 + 01 + 02 + vocabulary
Day 2: 03 + 08 + key stability drills
Day 3: 04 + 05 + 06 + 13 (state day)
Day 4: 07 + 09 + 10 (modules/refactors/troubleshooting)
Day 5: Team & governance (14–16) + capstone kickoff (22)

## Option 3: 2-week cohort (best for beginners)
- Run 1–2 labs/day with daily drills, review, and mini-projects.
